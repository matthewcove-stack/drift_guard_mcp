# Repository Documentation Contract

## Allowed Markdown Locations
- Allowed markdown paths are `docs/**/*.md`, plus root `README.md` and root `AGENTS.md`.
- No new doc folders without contract update.

## Canonical Locations
- canonical_phase_dir: `docs/phases`
- Phase docs must live in `docs/phases` and use `phase_*.md` naming.
- Canonical prompts location: `docs/prompts`.
- Runbooks location: `docs/runbooks`.

## Enforcement
- Any markdown outside allowed locations is a contract violation.
- Any filename containing `phase_` must live in `docs/phases`.
